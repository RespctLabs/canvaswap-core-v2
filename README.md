In-depth documentation on CanvaswapSwap is available at [docs.canvaswap.finance](https://docs.canvaswap.finance/).

# Local Development

The following assumes the use of `node@>=10`.

## Install Dependencies

`yarn`

## Compile Contracts

`yarn compile`

## Run Tests

`yarn test`
